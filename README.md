# Portfolio Template

This is a clean and modern **Portfolio Template** built with **HTML, CSS, and JavaScript**, inspired by the [Gridfolio Framer design](https://gridfolio.framer.website/).

The template is lightweight, fully responsive, and easy to customize—making it a perfect starting point for developers, designers, and creatives who want to showcase their work online.

### Features

* Minimal and elegant design
* Fully responsive layout for all devices
* Built with pure HTML, CSS, and JavaScript (no frameworks required)
* Easy to customize and extend
* Open-source and free to use

### Live Demo

👉 [View Portfolio Template](https://khairunnisa10.github.io/Portfolio-Template/)

### How to Use / Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/khairunnisa10/Portfolio-Template.git
   ```

2. **Navigate into the project folder**

   ```bash
   cd Portfolio-Template
   ```

3. **Open the template in your browser**
   Simply open the files (`index.html`, `about.html`, `contact.html`, `grids.html`, `resume.html`, `works.html`, `project-page.html`) in your preferred browser.

4. **Customize the content**

   * Replace text, images, and links with your own portfolio details.
   * Adjust styles in the `style.css` file.
   * Add interactions or enhancements in `script.js`.

5. **Deploy online**

   * You can easily publish this portfolio using **GitHub Pages**, **Netlify**, or **Vercel**.

Feel free to fork, modify, and use this template for your own projects. Contributions and suggestions are always welcome!
